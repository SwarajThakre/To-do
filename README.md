# ✅ To-Do List App

A simple and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript** that helps users manage daily tasks efficiently.

Users can add, complete, and delete tasks in a clean and responsive interface. The application demonstrates fundamental JavaScript concepts such as **DOM manipulation, event handling, and local storage**.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 💾 Tasks saved using **Local Storage**
* 📱 Responsive design
* ⚡ Fast and lightweight application

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Logic and interactivity
